# AI-PYTHON

Build a neuro network from zero

# neruo(h1,h2, o) matrix
```   
   x1 x2 h1 h2  o  

x1 0  0  w1 w3  0  

x2 0  0  w2 w4  0  

h1 0  0  b1 0   w5  

h2 0  0  0  b2  w6  

o  0  0  0  0   b3  

#end of matrix  (b1 b2 b3 means bias of the neuro)
```

backpro algorithm

L(w1,w2,w3,w4,w5,w6,b1,b2,b3)


#usage
 ```
git clone https://github.com/Vialiceq/AI-PYTHON.git
```
